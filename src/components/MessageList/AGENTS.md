# src/components/MessageList

Transcript list components and part-rendering helpers used to display message history.

## Directories

- `parts/`: Built-in renderers for individual message-part types used in the transcript.

## Files

- `MessageList.tsx`: Renders the chat transcript and optional loading state.
- `index.ts`: Barrel export for the transcript list component.
