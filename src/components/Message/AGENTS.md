# src/components/Message

Message-level UI for the chat transcript, including message bubbles, loading state, action buttons, and part formatting hooks.

## Directories

- `formatters/`: Renderers for individual message-part types.

## Files

- `LoadingIndicator.tsx`: Animated loading indicator shown for assistant placeholder messages.
- `Message.tsx`: Renders a single transcript message bubble, timestamp, and optional actions.
- `MessageButtons.tsx`: Renders the button row attached to a message.
- `index.ts`: Barrel export for the message bubble component.
