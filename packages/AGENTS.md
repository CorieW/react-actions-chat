# packages

Workspace packages published alongside the main `react-actions-chat` library.

## Directories

- `react-actions-chat-llms/`: Companion package that adds backend-routed text-generation helpers for chat flows.
- `react-actions-chat-recommended-actions/`: Companion package that adds query-driven and vector-search-backed recommended actions.
- `react-actions-chat-support/`: Companion package that adds reusable support-desk flows for customer and admin chat experiences.

## Files

- None.

## Writing Rules

- Each package should have a corresponding example in the `examples` directory (found in the root folder of project).
- Follow inherited AGENTS.md guidance when applicable.
