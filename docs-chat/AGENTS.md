# docs-chat

Shared source for the docs homepage chat frontend and its Firebase backend.

## Directories

- `backend/`: Firebase function source, runtime manifest, and generated retrieval assets for the docs assistant backend.
- `frontend/`: React-based homepage chat implementation that the VitePress theme mounts through a thin wrapper.

## Files

- None.

## Writing Rules

- Regenerate files in `backend/generated/` with the repo scripts instead of hand-editing them.
- Follow inherited AGENTS.md guidance when applicable.
