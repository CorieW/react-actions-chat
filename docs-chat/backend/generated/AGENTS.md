# docs-chat/backend/generated

Generated retrieval assets consumed by the docs assistant backend, including JSON outputs such as `docsAssistantCorpus.json` and `docsAssistantEmbeddings.json`.

## Directories

- None.

## Files

- `.gitkeep`: Keeps the generated assets directory tracked before JSON artifacts exist locally.
- `docsAssistantCorpus.json`: Generated docs chunk corpus committed for deterministic retrieval builds and CI sync checks.

## Writing Rules

- Regenerate JSON artifacts in this directory with the repo scripts instead of hand-editing them.
- Update this AGENTS.md when the tracked files in this directory or the generated artifact conventions change.
