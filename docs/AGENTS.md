# docs

VitePress source for the public documentation site, including guides, references, and custom theme code.

## Directories

- `.vitepress/`: VitePress configuration and custom theme code for the docs site.
- `components/`: Component-focused documentation pages for the main package APIs.
- `guides/`: Narrative guides for building chat flows and customizing behavior.
- `reference/`: API reference pages for published package surfaces.
- `stores/`: Reference pages for exported store hooks and state helpers.
- `sub-packages/`: Documentation for companion packages published from this workspace.
- `types/`: Reference pages for exported TypeScript types.

## Files

- `.gitignore`: Ignore rules for VitePress caches and generated docs output.
- `examples.md`: Guide to the runnable example apps and what each demo shows.
- `getting-started.md`: Getting-started guide for installing and rendering the chat component.
- `index.md`: Landing page for the documentation site.
