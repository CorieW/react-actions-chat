## Packages

- Core package: `actionable-support-chat`
- Companion package: `actionable-support-chat-recommended-actions`

Use the core package when you want a chat UI with buttons, confirmations, and shared-input flows. Add the companion package when you want query-driven or vector-search-backed recommendations that turn a user request into the next best actions.
