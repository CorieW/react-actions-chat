# docs/.vitepress

VitePress site configuration and theme overrides for the documentation site.

## Directories

- `theme/`: Custom theme assets and components used by the documentation site.

## Files

- `config.ts`: VitePress site configuration for navigation, theme metadata, and base-path handling.

## Writing Rules

- When adding, removing, or renaming docs pages or sections, update `config.ts` navigation and sidebar entries in the same change so the site keeps every page discoverable.
- Follow inherited AGENTS.md guidance when applicable.
