Actionable Support Chat is a modern, flexible React chat component library that enables you to build interactive customer support experiences with ease.

## Features

### 🎯 Interactive Action Buttons
Add actionable buttons to your messages that users can click to perform operations directly within the chat interface.

### ✅ Confirmation Requests
Request user confirmation for critical actions like account deletion with built-in confirmation workflows.

### 📝 Input Collection
Collect user input directly in the chat with validation support for various input types (text, email, password, etc.).

### 🎨 Theming Support
Fully customizable theming system with built-in light and dark themes, or create your own custom themes.

### 💾 State Management
Built on Zustand for efficient and predictable state management.

### 🚀 TypeScript First
Written in TypeScript with full type safety and excellent IntelliSense support.

## Use Cases

- **Customer Support Chats**: Build interactive support conversations with action buttons
- **Account Management**: Allow users to update settings, change passwords, and manage preferences within chat
- **Guided Workflows**: Create step-by-step processes with confirmation and input collection
- **Chatbots**: Build conversational interfaces with interactive elements

## Architecture

The library is built with modern React practices:

- **React 19** with latest hooks and patterns
- **Zustand** for lightweight state management
- **Tailwind CSS 4** for styling
- **TypeScript** for type safety
- **Vite** for fast development and builds
