# examples/uploads

Runnable upload-focused example app that showcases optional file uploads, file validators, and the built-in image/file message parts.

## Directories

- None.

## Files

- `App.tsx`: Main example UI and upload-flow orchestration for the uploads example.
- `index.css`: Example-specific visual design for the uploads demo shell and layout.
- `index.html`: Vite HTML entry point for the uploads example.
- `main.tsx`: React bootstrap for the uploads example application.
- `package.json`: Example package manifest and dev scripts.
- `tsconfig.json`: TypeScript configuration for the uploads example workspace package.
- `vite.config.ts`: Vite configuration for local development and production builds of the uploads example.
