# examples/login

Login-focused example app that demonstrates request-input buttons, confirmation flows, and a richer multi-step support experience.

## Directories

- None.

## Files

- `App.tsx`: Main example UI and flow orchestration for the sign-in assistant.
- `index.css`: Example-specific visual design layered on top of the shared chat styles.
- `index.html`: Vite HTML entry point for the example app.
- `loginDemo.ts`: Local demo account data and helper functions that keep the login flow deterministic.
- `main.tsx`: React bootstrap for the example application.
- `package.json`: Example package manifest and dev scripts.
- `tsconfig.json`: TypeScript configuration for the example workspace package.
- `vite.config.ts`: Vite configuration for local development and production builds of the example.
